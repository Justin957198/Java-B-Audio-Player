<h1>Java Based Audio Player.</h1> <br>
  This program is a very simple music /  sound player in the format of an MP3 player. 
  It utilizes java swing for UI elements such as forward, backward, pause, etc. 
  Once ran once a folder is created with a text file of instructions and an inner folder for song storage then songs can be added (only .wav). <br>
<h1>Instalation.</h1> <br>
  Make sure Java runtime is installed on the computer, clone the ripository and build the .java file with an IDE to run. <br>

  Once ran and songs have been added the buttons such as |> || R play, pause, and restart respectivly can be used. The four buttons on the bottom with arrows are used for       playlist traversal the buttons with the sort tails only going forward and backward by one song. The buttons with the long tails besides them will go forward and backwards by a random amount. The three text fields will display the song author the song number and the song title (keep in mind this only works for song file names in the format song author - song name.wave). Lastly the playlist button will list all the songs currently in the MP3 directory in another window. <br>
